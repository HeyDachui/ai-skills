# 私人智能分流网络教程

一份面向具备基础 Linux 与 Windows 运维能力的个人用户的教程：使用海外 VPS、WireGuard 与 Windows TUN 规则，把国内访问保持本地直连，将其他公网流量经由私人加密通道处理。

## 这是什么

- 面向少量具名设备的私人网络基础设施教程。
- 说明服务器加固、每设备独立身份、DNS 同路由、Windows 规则分流、验收与回滚。
- 不附带源码、Windows 安装包、服务器脚本、配置文件、设备 Profile 或第三方二进制。

## 这不是什么

- 不是公开代理、商业 VPN、多租户服务或账号共享方案。
- 不保证平台接受某个出口 IP，也不保证长期稳定。
- 不提供任何真实服务器、端口、密钥、密码、Cookie、API Key 或设备身份。

## 使用前请注意

请自行核对当地法律、VPS 服务商条款与目标平台规则。不要将本教程用于违法活动、侵权、绕过平台限制、公开转售网络服务或向他人共享私钥／设备 Profile。
