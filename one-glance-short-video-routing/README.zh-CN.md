# 一眼分流法：短视频下载转写与意图分流 Skill

版本：v1.0.0

## 简介

“一眼分流法”是一个用于外部短视频素材进入本地系统后的入口型 Skill。

它不只是下载和转写视频，而是先判断用户为什么保存这条素材：是参考内容、表现形式、画面风格、提示词、工作流、账号策略、音频节奏，还是只是暂时留存。

它帮助 Codex 对外部素材完成去重、转写、主辅材料分层、参考意图标注和第一轮分析分流，并决定素材应该跳过、留存、轻量分析、深度分析、交给其他线程，还是作为外部候选资产继续处理。

## 核心价值

- 避免把所有素材都机械总结成“内容摘要”。
- 避免把画面风格、账号策略、提示词、工作流混成一种分析口径。
- 避免把外部素材未经审核地直接吸收进内部规则或正式资产。
- 帮助多线程协作时快速定位素材、说明参考意图、减少交接成本。

## 核心原则

外部素材默认仍是外部资产。  
它可以被分析、借鉴和转译，但进入内部规则、正式 Skill、Prompt 面板或正式知识资产前，必须经过本地适配、边界确认和用户审核。

## 适用场景

- 抖音或短视频链接下载、转写、去重。
- 账号批量内容进入素材库后的第一轮分流。
- 判断某条视频该看内容、形式、画面、提示词、工作流、账号策略还是音频节奏。
- 给其他线程派发素材分析任务。
- 从外部素材中识别高价值候选资产。

## 不适用场景

- 直接生成内部正式规则。
- 直接把外部素材写入正式知识库。
- 替代专业领域深度分析 Skill。
- 在缺少逐字稿或视觉证据时强行深挖。
- 对用户明确要求跳过的素材继续分析。
