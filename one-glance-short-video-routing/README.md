# One-Glance Short Video Routing Skill

Version: v1.0.0

## Overview

One-Glance Short Video Routing is an entry-layer skill for external short-video materials entering a local workspace.

It does more than download and transcribe videos. It first identifies why the user saved the material: content reference, presentation style, visual style, prompt or generation method, workflow, account strategy, audio rhythm, or simple retention.

The skill helps Codex deduplicate materials, transcribe spoken videos, separate primary media from auxiliary files, tag reference intent, and route each item into skip, retain, light analysis, deep analysis, handoff, or external candidate asset processing.

## Core Value

- Avoid reducing every material to a generic content summary.
- Avoid mixing content, presentation, visuals, prompts, workflows, account strategy, and audio rhythm into one analysis mode.
- Prevent external materials from being absorbed into internal rules or formal assets without review.
- Help multi-thread collaboration by making each material easier to locate, explain, and hand off.

## Core Principle

External materials remain external by default.  
They may be analyzed, referenced, and translated into reusable drafts, but they must not become internal rules, formal skills, prompt panels, or formal knowledge assets without local adaptation, boundary review, and user approval.

## Use Cases

- Downloading, transcribing, and deduplicating short-video links.
- Routing creator batches or account downloads after they enter a local material library.
- Deciding whether a video should be analyzed for content, presentation, visual style, prompts, workflow, account strategy, or audio rhythm.
- Preparing handoff packages for other threads.
- Identifying high-value external candidate assets.

## Non-Goals

- Directly generating internal formal rules.
- Writing external materials into a formal knowledge base.
- Replacing domain-specific deep analysis skills.
- Forcing deep analysis without transcript or visual evidence.
- Analyzing materials the user explicitly asked to skip.
