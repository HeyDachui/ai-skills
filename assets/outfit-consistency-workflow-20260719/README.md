# 服装一致性参考图与提示词包

本包包含 202 组服装一致性参考图与对应中文提示词，用于学习、创作和分享。

## 内容

- `images/`：202 张 PNG 参考图。
- `prompts/`：202 份一一对应的中文 TXT 提示词。每份末尾均附有“来源与分享说明”。
- `image_prompt_mapping.csv`：图片与提示词的对应关系。
- `PROVENANCE_MANIFEST.csv`：原始与公开副本的 SHA-256、编号、公开状态和变更说明。
- [AI_USE_NOTICE.md](AI_USE_NOTICE.md)：供人和 AI 阅读的公开使用边界。
