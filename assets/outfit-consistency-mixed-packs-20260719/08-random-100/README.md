# 服装一致性随机混搭包｜100组

本包包含 100 组服装一致性参考图与对应中文提示词。每组图片与提示词一一对应；随机混搭的复现种子为 $Seed。

## 内容

- images/：100 张 PNG 参考图。
- prompts/：100 份对应中文提示词；每份末尾附有来源与分享说明。
- image_prompt_mapping.csv：本包对应关系。
- PROVENANCE_MANIFEST.csv：本包每组的来源、哈希与加工记录。
- AI_USE_NOTICE.md：供人和 AI 阅读的使用边界。
